OUR OWN ZETA

Our Own Zeta is a project that will simulate the evolution of civilizations over time, offering players the chance to manage various resources, discover new technologies, and develop key figures that will help the civilization grow.

As civilizations advance, they will face random events such as wars, natural disasters, and breakthroughs that will change the course of their history.

Technologies

Java has been used as the core backend of the project. Backend uses a Rest API to manage interaction between the front end and the back end

Spring Boot powers the backend structure of the game. It simplifies setup and helps expose data through secure and scalable RESTful services to the frontend.

React is used optionally for the game's interface. It enables a responsive UI to display civilization status, events, and key information in real-time.

TypeScript may be used on the frontend for type safety. It enhances code quality and reduces bugs when handling game state and user interaction.
